## Method 1: Two Stacks (TLE)
Time Complexity: </br>
Push(): O(1) </br>
Pop(): O(1) </br>
Top(): O(1) </br>
peekMax(): O(1) </br>
popMax(): O(n) </br>

## Method 2: Stack + Priority Queue (TLE)
Time Complexity: </br>
Push(): O(logn) </br>
Pop(): O(logn) </br>
Top(): O(1) </br>
peekMax(): O(1) </br>
popMax(): O(n) </br>
